# ANVORA AI Operating System

## Project Identity

Anvora is a premium AI Operating System for founders, traders, and businesses.

The application must feel like an executive workspace rather than a typical dashboard.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- React Router
- Lucide React

## Design Principles

- Apple-quality UI
- Executive dark theme
- Minimalist
- Premium spacing
- Reusable components
- Accessibility first
- Mobile responsive

## Architecture

Use feature-first architecture.

src/
  app/
  components/
  features/
  layouts/
  lib/

Never generate code outside this architecture unless explicitly instructed.

## Coding Standards

- Strong TypeScript typing
- Functional React components
- Reusable UI
- Clean separation of concerns
- Avoid duplicated logic
- Avoid unnecessary dependencies

## Output Rules

When implementing features:

1. Explain the approach briefly.
2. Generate complete production-ready code.
3. Keep components modular.
4. Reuse existing shadcn components.
5. Prefer maintainability over shortcuts.
